This website serves to help MHS students know if each school day is a "A" or "B" day. This website was created by Ethan Hampton when he realized that no one knew what day it was, and it would be helpful to have a website to tell you about it.
This website is Progressive Web Application. This means that you can add it to your home screen on mobile and it will work offline.
